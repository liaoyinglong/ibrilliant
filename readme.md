## ibrilliant 公共库

git 地址 http://192.168.70.56:8088/hopex/ibrilliant

### install

```bash
# dependencies
yarn add @ibrilliant/hooks @ibrilliant/observable @ibrilliant/utils
# devDependencies
yarn add -D @ibrilliant/locale-cli @ibrilliant/sw2rx @ibrilliant/umi-preset
```
